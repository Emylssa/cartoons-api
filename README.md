# Cartoons 90s & 2000s - Front-End Project

Projeto estático que consome a API do Rick and Morty e gera cards dinamicamente via JavaScript.

## Como usar
1. Abra `index.html` em um navegador (não necessita servidor).
2. A aplicação usa `fetch` para obter personagens da API pública: https://rickandmortyapi.com/
3. É possível pesquisar e filtrar por status, além de carregar mais personagens.

## Arquivos
- index.html
- styles.css
- script.js

## Observações
- A API é pública e não requer token.
- Caso precise publicar no GitHub Pages, basta enviar estes arquivos ao repositório.
